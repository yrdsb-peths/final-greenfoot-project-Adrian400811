# Final Greenfoot Project
This project is a horrible spinoff of diep.io, made with Greenfoot.

## Project Status
The project has a working playable character, destroyable objects, and health and experience system.  
The project is currently having no goal.  
There is no level limit.  

## Planning
? we don't plan here

## How to play
wasd to move  
move mouse to turn  
space to shoot  
e to toggle autoshoot  

## Goal
Freeplay: no ending
Challenge mode: reach lv10 with shortest time

## Install
1. Install Greenfoot on your machine
2. Clone the repository
3. Double-click on project.greenfoot
